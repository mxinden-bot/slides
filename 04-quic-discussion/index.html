Evolving HTTP/3 & QUIC

Growing usage beyond 30% of the Internet?

HTTP Workshop 2026 · discussion  ·  Max Inden · Mozilla

Is HTTP/3 really “beyond 30%”?

The neutral number has plateaued

  • Cloudflare request share sits at ~21%, flat since a ~28% peak in 2023.
  • “>30%” holds only for advertised support (W3Techs ~39%) or hyperscaler first-party traffic (Google ~42%, Meta ~75%).
  • Requests vs connections is a real confound: HTTP/3 multiplexes many requests per connection.
  • “QUIC is not Quick Enough over Fast Internet”: up to -45% throughput vs HTTP/2 above ~500 Mbps.

QUIC v2: built to be deployed, barely deployed

  • RFC 9369 is functionally identical to v1: only crypto salts, labels and codepoints differ (anti-ossification).
  • Measured deployment: < 0.003% of QUICv1 domains; < 0.013% grease the QUIC bit.
  • No feature payoff, so no individual incentive. neqo supports v2 (compatible upgrade); quinn does not.
  • Is greasing (RFC 9287) enough that v2 is unnecessary?

Alt-Svc → HTTPS RR: the upgrade path

  • Alt-Svc is learned after a slower HTTP/2 connection: the first visit never uses HTTP/3.
  • Firefox Nightly: 22% of connections learn h3 only via Alt-Svc, a wasted round trip.
  • HTTPS records (RFC 9460) are ~15% adopted, and ~70% of those are one CDN’s defaults.
  • ECH depends on the HTTPS record: it is now a privacy lever, not just a perf one.

Do we deprecate Alt-Svc, or bridge it (draft-thomson-httpbis-alt-svcb)?

Optimistic QUIC, Happy-Eyeballs style

  • Race QUIC speculatively on the first connection, before any h3 signal.
  • curl (Aug 2025): start QUIC, then TCP after 200 ms, first wins.
  • Risk: on the 1–5% of UDP-hostile networks, an optimistic attempt is strictly slower (timeout + TCP + TLS).
  • The right racing delay is unsettled and network-dependent.

Open questions

  • Which denominator should we rally around, and is the plateau since 2023 the real worry?
  • Is the fast-internet throughput regression the ceiling on QUIC growth?
  • Can QUIC v2 ever escape its coordination trap? Does a major library shipping no v2 matter?
  • Deprecate Alt-Svc for HTTPS RR, or bridge? Is ECH the lever that finally forces HTTPS-RR publication?
  • Default posture for optimistic QUIC, and who eats the fallback tax on UDP-hostile networks?
