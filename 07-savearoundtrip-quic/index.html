Alt-Svc only

HTTP/3 only on the second connection.

HTTPS record

HTTP/3 on the first connection.

35% of Firefox connections learn h3 only via Alt-Svc (GLAM): a wasted round trip. An HTTPS record (RFC 9460) fixes it and also carries IP hints and ECH, yet few publish one beyond a single CDN’s automatic default (Dong et al., IMC 2024). Publish one with alpn="h3".

QUIC adoption: run v2, not just v1

  • QUIC v2 (RFC 9369) is functionally identical to v1: only salts, labels, and codepoints differ, purely anti-ossification.
  • Firefox Nightly: v1 ~100%, v2 ~0% of version-labeled handshakes (http_3_quic_version). Wider scans agree: v2 on <0.003% of QUICv1 domains (IMC 2024).
  • Implementing v2 is common, yet deploying it is rare (next slide).
  • v2 buys no new feature, only anti-ossification, so nobody moves first. Every visible deployment adds pressure on the holdouts.

Probe any origin’s live v1/v2 support at savearoundtrip.com: a cold h3 handshake that reports the versions offered.
